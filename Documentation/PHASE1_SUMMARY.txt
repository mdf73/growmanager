ENDPOINTS API (Backend - FastAPI)
================================================================================

BREEDERS
  GET    /api/breeders                # Tous les breeders
  GET    /api/breeders/{id}           # Breeder par ID
  POST   /api/breeders                # Créer breeder
  PUT    /api/breeders/{id}           # Modifier breeder
  DELETE /api/breeders/{id}           # Supprimer breeder

VARIETES
  GET    /api/varietes                # Toutes les variétés
  GET    /api/varietes/{id}           # Variété par ID
  POST   /api/varietes                # Créer variété
  PUT    /api/varietes/{id}           # Modifier variété
  DELETE /api/varietes/{id}           # Supprimer variété

GRAINES & PACKS
  GET    /api/graines                 # Toutes les graines
  GET    /api/graines/{id}            # Graine par ID
  POST   /api/graines                 # Créer graine
  PUT    /api/graines/{id}            # Modifier graine
  GET    /api/packs                   # Tous les packs
  GET    /api/packs/{id}              # Pack par ID
  POST   /api/packs                   # Créer pack
  PUT    /api/packs/{id}              # Modifier pack
  GET    /api/catalogue               # Catalogue complet avec stock

CULTURES
  GET    /api/cultures                # Toutes les cultures
  GET    /api/cultures/{id}           # Culture avec détails
  POST   /api/cultures                # Créer culture
  PUT    /api/cultures/{id}           # Modifier culture
  GET    /api/cultures/{id}/plants    # Plants d'une culture
  POST   /api/cultures/{id}/plants    # Créer plant
  GET    /api/cultures/plant/{id}     # Plant par ID
  PUT    /api/cultures/plant/{id}     # Modifier plant
  GET    /api/cultures/{id}/calendrier # Calendrier actions
  POST   /api/cultures/{id}/actions   # Créer action

STOCK
  GET    /api/stock                   # Tout le stock
  GET    /api/stock/{id}              # Item stock par ID
  POST   /api/stock                   # Créer stock
  PUT    /api/stock/{id}              # Modifier stock

EXTRACTIONS
  GET    /api/rosin                   # Extractions rosin
  POST   /api/rosin                   # Créer extraction rosin
  GET    /api/rosin/stats             # Stats extraction
  GET    /api/hash                    # Extractions hash
  POST   /api/hash                    # Créer extraction hash

DASHBOARD
  GET    /api/dashboard               # Stats globales

HEALTH
  GET    /health                      # Vérification santé API

  • IceOBag              - Sacs ice-o-bator
  • Engrais              - Engrais et additifs

CULTURE
  • Culture              - Cycles de culture
  • Plant                - Plants individuels (NEW)
  • ActionCalendrier     - Actions timeline (NEW)
  • CultureGraine        - Liaison Culture ↔ Graine

  • CultureLampe         - Liaison Culture ↔ Lampe
  • CulturePot           - Liaison Culture ↔ Pot

RECOLTE
  • Recolte              - Récoltes

STOCK
  • Stock                - Stock de fleurs séchées

EXTRACTIONS
  • RosinExtraction      - Extractions rosin
  • HashExtraction       - Extractions hash
