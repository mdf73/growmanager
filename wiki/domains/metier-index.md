# Vue métier — GrowManager

Vue organisée par fonctionnalité/domaine de culture plutôt que par couche technique (api/database/frontend). Chaque page ci-dessous décrit **ce que fait l'app** pour ce sujet, puis renvoie vers le détail technique si besoin.

- [[domains/metier-cultures]] 🌱 — planification, suivi, séchage/curing, historique, calendrier, comparaison, classement variétés
- [[domains/metier-graines]] 🌾 — catalogue de graines, breeders, croisement/génétique maison
- [[domains/metier-recettes-sol-vivant]] 🧪 — TCO, LSO, arrosage, fermentation, réamendement, engrais, suivi sol vivant
- [[domains/metier-stock-extractions]] 📦 — stock produit fini, extractions rosin/hash
- [[domains/metier-equipement-espaces]] 🔧 — espaces de culture, matériel, paramétrage
- [[domains/metier-capteurs]] 🌡️ — température, humidité, VPD, capteurs Govee
- [[domains/metier-photos]] 📸 — galerie photo culture/plante
- [[domains/metier-mobile-app]] 📱 — application Android Capacitor

Pour la documentation technique (modèles DB, endpoints API, composants frontend, décisions d'archi), voir [[index]] — le wiki technique complet.
